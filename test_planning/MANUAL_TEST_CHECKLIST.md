# Manual Test Checklist

این چک‌لیست برای تست دستی کل سیستم است. هر مورد را بعد از انجام با `[x]` علامت بزنید و اگر مشکلی بود در انتهای فایل ثبت کنید.

---

## Phase 1: Setup & Auth
- [ ] نصب وابستگی‌ها بدون خطا (`npm install`)
- [ ] بررسی `.env` با `env.example` و وجود کلیدها
- [ ] `npm run build` بدون خطا
- [ ] `npm run lint` بدون خطا یا با خطای قابل‌قبول
- [ ] `npx prisma validate`
- [ ] `npx prisma migrate status` همگام
- [ ] `npm run prisma:seed` موفق
- [ ] OTP ارسال کد بدون خطا
- [ ] لاگین admin با `admin/admin123`
- [ ] بررسی نقش‌ها و دسترسی `/admin`

## Phase 2: Product & Inventory
- [ ] ایجاد محصول جدید از UI یا API
- [ ] مشاهده محصول و جستجو
- [ ] تغییر موجودی محصول
- [ ] حذف محصول
- [ ] ثبت رکورد Inventory و بررسی تاثیر روی stock
- [ ] تغییر قیمت طلا و بررسی تاثیر روی محصولات وابسته

## Phase 3: Sales & Orders
- [ ] افزودن محصول به سبد خرید (guest)
- [ ] مشاهده سبد خرید
- [ ] ثبت سفارش (manual)
- [ ] کاهش موجودی بعد از سفارش
- [ ] صدور فاکتور برای سفارش
- [ ] کنسل سفارش و بازگشت موجودی

## Phase 4: CRM & Messaging
- [ ] ایجاد مشتری جدید
- [ ] مشاهده علایق مشتری
- [ ] ثبت بدهی/طلب
- [ ] مشاهده لیست بدهی‌ها
- [ ] ارسال پیام دستی
- [ ] اجرای پیام‌رسانی خودکار (اگر فعال است)

## Phase 5: Special Features
- [ ] آپدیت قیمت طلا و ثبت تاریخچه
- [ ] ایجاد Scrape Target
- [ ] اجرای Scrape (Mock)
- [ ] ایجاد پست بلاگ و بررسی slug یکتا

## Phase 6: Admin & Reports
- [ ] صحت کارت‌های داشبورد
- [ ] نمودار فروش ماهانه
- [ ] گزارش سرمایه در گردش
- [ ] پیش‌بینی فصلی (Analytics)
- [ ] تغییر و ذخیره BusinessSettings

## Phase 7: E2E Final
- [ ] مسیر خرید کامل کاربر (از ورود تا سفارش)
- [ ] مشاهده سفارش در پنل ادمین
- [ ] تغییر وضعیت سفارش و صدور فاکتور
- [ ] بررسی ریسپانسیو موبایل
- [ ] Lighthouse (Home + Product)
